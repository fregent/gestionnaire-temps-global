# ⏰ Gestionnaire Temps Global

Une application web simple, rapide et responsive pour gérer les fuseaux horaires, un chronomètre et un minuteur — tout en temps réel.

## 🎯 Fonctionnalités

### 🌍 Horloge Multi-Fuseaux
- Affichage simultané de l'heure dans plusieurs fuseaux horaires
- Ajout/suppression dynamique de villes
- Mise à jour en temps réel chaque seconde
- Support de 17 villes majeures (Paris, Tokyo, New York, Sydney, etc.)

### ⏱️ Chronomètre
- Démarrage/arrêt du chronomètre
- Fonction de tour (lap) pour enregistrer des temps intermédiaires
- Réinitialisation complète
- Précision au centième de seconde

### ⏲️ Minuteur
- Configuration personnalisée (minutes et secondes)
- Démarrage/arrêt du minuteur
- Alerte sonore à la fin du décompte
- Réinitialisation simple

## 📋 Prérequis

- Un navigateur moderne (Chrome, Firefox, Safari, Edge)
- Aucune dépendance externe requise (HTML5, CSS3, JavaScript vanilla)
